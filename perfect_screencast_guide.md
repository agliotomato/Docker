### 🎬 Scene 2: Docker 설정 검증 및 컨테이너 가동 확인 (0:25 - 0:50)
* **주요 화면**: 좌측 **[EC2 SSH 터미널]** 유지
* **실행 명령어 (각각 마우스 우클릭으로 붙여넣어 순서대로 실행하세요)**:
  ```bash
  cat docker-compose.yml
  ```
  ```bash
  docker ps
  ```
* **마우스 연출**:
  - `docker-compose.yml` 출력 시: `ports: 80:8000` 및 `restart: always` 라인 마우스 드래그 강조.
  - `docker ps` 결과 출력 시: `STATUS` 열의 `Up` 부분과 `PORTS` 열의 `0.0.0.0:80->8000/tcp` 영역을 마우스 커서로 강조.
* **목소리 낭독 대본 (VO)**:
  > **"접속에 성공한 서버 터미널 환경에서 `cat docker-compose.yml`을 실행하여 외부 접속용 80번 포트가 컨테이너 내부의 8000 포트로 완벽히 매핑되었고, 자동 재시작을 보장하는 `restart: always` 정책이 기재된 것을 확인하실 수 있습니다. 이어서 `docker ps` 명령어를 수행하면 보시는 바와 같이 컨테이너가 정상적으로 가동되어 외부 80번 포트를 통해 접속 대기 중인 상태를 보여줍니다."**

  ```json

    "semester": "1",
    "grade": "A+"
  ```
