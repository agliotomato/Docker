### 🎬 Scene 2: Docker 설정 검증 및 DB 파일 교차 확인 (0:25 - 1:00) [★ DB 교차 검증 탑재!]
* **주요 화면**: 좌측 **[EC2 SSH 터미널]** 유지
* **실행 명령어 (각각 붙여넣어 연달아 실행하세요)**:
  ```bash
  cat docker-compose.yml
  cat courses.json
  docker ps
  ```
* **마우스 연출**:
  - `docker-compose.yml` 출력 시: `ports: 80:8000` 및 `restart: always` 라인 마우스 드래그 강조.
  - `courses.json` 출력 시: **`자료구조`** 및 **`빅데이터프로그래밍`** 텍스트 부분을 드래그하여 원본 데이터임을 강조.
  - `docker ps` 결과 출력 시: `STATUS` 열의 `Up` 부분과 `PORTS`의 `80->8000/tcp` 매핑 영역 드래그 강조.
* **목소리 낭독 대본 (VO)**:
  > **"접속에 성공한 서버 터미널 환경에서 `cat docker-compose.yml`을 실행하여 외부 80 포트 연동과 `restart: always` 정책이 기재되어 있음을 먼저 보여드립니다.**
  > 
  > **이어서 `cat courses.json` 명령어를 입력하여 서버 내부에 저장된 원본 데이터베이스 파일을 출력합니다. 보시는 바와 같이 자료구조와 빅데이터프로그래밍 등 제출 규격에 맞는 원본 교과목 데이터가 안전하게 저장되어 있습니다.**
  > 
  > **마지막으로 `docker ps`를 구동하면, 보시는 것과 같이 `docker-web` 컨테이너가 `Up` 상태로 활발히 동작하고 있으며 외부 80번 포트가 컨테이너 내부의 8000 포트로 정확하게 포워딩되어 통신을 대기 중인 상태를 보여줍니다."**
