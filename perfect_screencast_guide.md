* **실행 명령어 (복사하여 터미널에 우클릭 붙여넣기 하세요)**:
  ```powershell
  ssh -i C:\Users\skht0\OneDrive\Downloads\labsuser.pem ubuntu@34.235.213.75
  ```
* **마우스 연출**: 로그인에 성공하여 터미널 주소인 `ubuntu@ip-172-31-31-130:~$`가 정상 출력되면, 해당 프롬프트 라인을 마우스 커서로 가볍게 한번 긁어 줍니다.
* **목소리 낭독 대본 (VO)**:

### 🎬 Scene 2: 프로젝트 디렉터리 이동 및 3대장 설정 검증 (0:25 - 1:00) [★ 3대장 명령어 콤보 완벽 융합!]
* **주요 화면**: 좌측 **[EC2 SSH 터미널]** 유지
* **실행 명령어 (각각 마우스 우클릭 복사하여 연달아 실행하세요)**:
  ```bash
  cd Docker/
  ```
  ```bash
  cat docker-compose.yml
  ```
  ```bash
  cat courses.json
  ```
  ```bash
  docker ps
  ```
* **마우스 연출**:
  - `cd Docker/` 입력 시: 디렉터리가 `~/Docker`로 진입하여 터미널 경로 프롬프트가 전환되는 것을 보여줍니다.
  - `docker-compose.yml` 출력 시: `ports: 80:8000` 및 `restart: always` 라인 마우스 드래그 강조.
  - `courses.json` 출력 시: **`자료구조`** 및 **`빅데이터프로그래밍`** 등 원본 JSON 데이터 목록 드래그 강조.
  - `docker ps` 결과 출력 시: `IMAGE` (docker-web), `STATUS` (Up), `PORTS` (80->8000/tcp) 영역을 마우스 커서로 강조.
* **목소리 낭독 대본 (VO)**:
  > **"접속에 성공한 서버 터미널 환경에서, 먼저 `cd Docker/` 명령어를 실행하여 소스코드가 기동 중인 프로젝트 폴더로 이동합니다. 이어서 `cat docker-compose.yml`을 실행하여 외부 접속용 80번 포트가 컨테이너 내부의 8000 포트로 매핑되었고, 자동 재시작을 보장하는 `restart: always` 정책이 기재되어 있음을 보여드립니다.**
  > 
  > **다음으로 `cat courses.json` 명령어를 통해 서버 내부의 원본 데이터베이스 내용을 출력하여, 자료구조와 빅데이터프로그래밍 과목이 원본 데이터셋에 포함되어 있음을 확인합니다.**
  > 
  > **마지막으로 `docker ps` 명령어를 수행하여 런타임 가동 현황을 증명하겠습니다. 보시는 것처럼 빌드 완료된 `docker-web` 이미지를 기반으로 생성된 컨테이너가 현재 활성화 상태인 `Up` 상태로 완벽히 가동 중이며 외부 80포트 통신 규격이 실시간으로 활성화되어 있음을 완벽하게 입증해 줍니다."**
